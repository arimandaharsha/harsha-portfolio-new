## üèÅ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/arimandaharsha/harsha-portfolio.git
   cd harsha-portfolio
   ```
2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   ```
4. **Open [http://localhost:3000](http://localhost:3000) to view the site.**
